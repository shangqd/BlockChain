# 拜占庭问题。

1 eos 中不可逆块是这样算出来的

不可逆块 =  12 * 21 + 12 * 21 / 3
第一项： 确定签名集合
第二项： 确定集合后，对当前链的签名人员的收集，不能小于三分之一，（叛徒也可以签1/3的长度，因为有三分之一的叛徒，叛徒分完叉后，可以把正常其他未签名的节点给
忽悠过来，其他未签名的节点可能还没有看到正确签名的长度，因此链就产生了一次回滚）（当然了签名者都不是叛徒）
         不签名的企图分叉者，才是叛徒。
         
第二种解释
每个签名者都会说一个块，已经不可逆，当然是2/3的已经签名，当有2/3的人说他不可逆的时候，他就真的不可逆了，
两者在解释上是等效的，具体有如下公式。
2 / 3 + 2 / 3 = 1 + 1 / 3

# 如果是容错1/2的叛徒呢
1 / 2  +  1 / 2 = 1 + 1 / 2

根据上面的公式，两种解释出现了不一致性，必定有一个是错误的，第二种跟直觉冲突，显然第二种是错误的。

/*
因此完善第一种解释
在集合确定的时候，最多有三分之一的人退出，有三分之一的人加盟，然后他们用各种方法忽悠了这2/3的人，加上这1/3的叛徒
一起构成了一个新的集合，因此回滚的再次向上推了一个集合周期。（诚实的出块者不可能在同一高度签两次名）。
解释失败，递推下去没完没了，整个链都可以回滚。
才有如下解释
*/

可以简单的想成集合确实阶段+加签名确定阶段.

新的签名产生，也同时改变签名集合。
如果是新加入的人签名，应该不会让不可逆快正常的增加，因为新加入的人不是集合确定人员之列。
这个通过eos 确认下便可知道

如果有1/3之一的坏蛋，他们可以对集合进行填充僵尸，然后对僵尸签名，然后（所说的第二阶段）继续签名，同样保持了当前高度。
如果还继续向前走的话，签名高度将会变低，根据最长链原则，本链将会被吃掉，因此得出支持第一种说法的结论（新加入的签名者不影响高度）
